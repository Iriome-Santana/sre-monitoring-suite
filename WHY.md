# Why this project exists

Este proyecto no pretende reemplazar soluciones como Prometheus
o Datadog.

Fue diseñado para:
- Entender cómo funciona el monitoreo desde cero
- Implementar alerting sin dependencias externas
- Practicar gestión de estado y alert fatigue
- Simular responsabilidades reales de un SRE en entornos pequeños

Decidí usar herramientas simples para priorizar
entendimiento sobre complejidad.
